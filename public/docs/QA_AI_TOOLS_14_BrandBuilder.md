# QA — Brand Builder (P0)

**URL:** `/brand-builder` | **Kredity:** 5+ / komponent

## A. Prístup
1. `/brand-builder` → hero, "Build your brand" wizard alebo dashboard
2. Gate
3. Mobile

## B. Wizard
1. Step 1: Brand name, industry, target audience
2. Step 2: Brand values (vyber 3-5 keywords)
3. Step 3: Style preferences (modern/classic/playful/luxury/eco)
4. Step 4: Competitor URLs (optional)
5. Submit → AI generuje brand foundation

## C. Logo
1. AI generuje 4 logo varianty
2. Klik favorite → save
3. Regenerate → ďalšie kredity
4. Edit colors, text, icon → mini editor
5. Download SVG/PNG/PDF (transparent BG)
6. Variants: full, icon-only, wordmark, monogram

## D. Color palette
1. AI generuje primary, secondary, accent + neutrals (5-7 colors)
2. Hex values copy
3. Edit color picker
4. Accessibility check (WCAG contrast)
5. Export CSS variables / Figma tokens

## E. Typography
1. AI vyberie heading + body font pair
2. Preview text rendering
3. Swap suggestions (3-4 alternatives)
4. Font download links (Google Fonts)

## F. Brand voice
1. Tone guidelines (3-5 do's, 3-5 don'ts)
2. Sample copy (headline, body, CTA)
3. Edit/regenerate

## G. Brand book PDF
1. Klik "Generate Brand Book" → multi-page PDF
2. Cover, logo usage, colors, typography, voice, examples
3. Download PDF
4. Share link

## H. Apply across modules (cross-module integrácia)
1. Vytvor brand → otvor CreativeForge → brand colors auto-applied
2. Otvor Content Studio → brand fonts dostupné
3. Otvor Video Ad Generator → brand template

## I. My Brands
1. Multiple brands (per project)
2. Switch active brand
3. Edit, duplicate, delete

## J. Kredity
1. Full brand: 20-30 kreditov
2. Per komponent regenerate: 5
3. Insufficient → buy

## K. Multi-user
1. A vytvorí brand → private
2. Team feature (ak existuje) → invite B → B vidí + edituje
3. Share read-only link → klient vidí

## L. RLS
1. Brand data isolated per user/team
2. Brand book PDF storage signed

## M. i18n
1. Brand voice generated v zvolenom jazyku
2. UI translation
