# QA — AI Tattoo (P2)

**URL:** `/ai-tattoo` | **Kredity:** 4 / generácia

## A. Prístup
1. `/ai-tattoo` → hero, style gallery, "Create" CTA
2. Age gate (16+ ak je tattoo NSFW konotácia)
3. Mobile

## B. Generovanie
1. Vyplň concept prompt ("rose with skull")
2. Style: Traditional / Realistic / Tribal / Minimalist / Watercolor / Geometric / Japanese → 7 štýlov, každý generuje iný
3. Size dropdown: Small / Medium / Large → preview proporcie
4. Color toggle: B&W / Color
5. Placement: Arm / Back / Leg / Wrist / Chest → preview ON body (mockup)
6. Generate → výsledok

## C. Body placement preview
1. Upload vlastnú fotku tela → AI overlay tattoo
2. Vyber pre-set body mockup
3. Resize/rotate tattoo na fotke
4. Mirror flip

## D. Variácie
1. Generate variations (4) → 4 alternatívy
2. Save favorite → My Tattoos
3. Download PNG transparent BG

## E. Edge cases
1. NSFW prompt → blocked
2. Hate symbols → blocked
3. Prázdny prompt → disabled

## F. Kredity
1. 4 / gen, refund on fail
2. Insufficient → buy

## G. Multi-user
1. A vytvorí public design → B vidí v gallery
2. Share link

## H. My Tattoos
1. Save, delete, organize
2. Filter by style
