# QA — Home Designer (P1)

**URL:** `/home-designer` | **Kredity:** 4 / render

## A. Prístup
1. `/home-designer` → hero, "Design room" CTA, gallery
2. Gate
3. Mobile (limitovaný 3D editor)

## B. Tvorba projektu
1. Klik New → vyber room type: Living / Bedroom / Kitchen / Bathroom / Office / Outdoor
2. Set rozmery (m²)
3. Vyber style: Modern / Scandinavian / Industrial / Boho / Minimal / Classic / Japanese
4. Upload existing room photo (optional, AI transforms)
5. Generate → AI návrh

## C. 2D / 3D toggle
1. 2D floor plan view → editovateľný layout
2. Switch to 3D → walkthrough perspective
3. Camera controls: rotate, pan, zoom
4. Mobile touch controls

## D. Furniture placement
1. Catalog sidebar — kategórie (sofa, table, lamp, plant, art)
2. Drag furniture → place v room
3. Move, rotate, resize
4. Delete
5. Realistic furniture library (real brand products?)
6. Click furniture → details, buy link (affiliate?)

## E. Materials
1. Floor texture picker
2. Wall color/wallpaper
3. Ceiling
4. Real-time preview

## F. Render
1. Photorealistic render → kredity, loading
2. Multiple angles (4 views)
3. Day/night lighting toggle
4. Download HD PNG
5. Render 360° panorama

## G. AI suggestions
1. "Improve layout" → AI re-arranges
2. "Add greenery" → adds plants
3. Style transfer (current → new style)

## H. Kredity
1. 4 / render, multi-angle = 16
2. Insufficient → buy

## I. My Designs
1. Save projects
2. Duplicate
3. Share link
4. Export 3D file (OBJ/GLTF)

## J. Multi-user
1. A share design link → B vidí 3D view
2. Collaboration (ak je) → B edituje súčasne
3. Comments na design

## K. Performance
1. 3D render <60s
2. Browser nezamrzne pri komplexnom room
3. WebGL fallback
